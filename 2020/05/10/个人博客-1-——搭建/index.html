发表于2020-05-10更新于2020-05-11字数统计248阅读时长1分阅读次数

环境要求：node.js 和 npm
基于 Hexo博客框架

👉Hexo官方地址

1. 配置环境

安装node.js和npm
修改npm的镜像为阿里镜像，即按照cnpm

  • 安装cnpm
    cmd命令：
      npm install -g cnpm –registry=https://registry.npm.taobao.org
    安装测试：
      cnpm -v

2. 安装Hexo框架

2.1. 安装：cnpm install -g hero-cli

2.2. 验证安装：hexo -v

3. 初始化Hexo博客

3.1. 初始化hexo项目：hexo init

3.2. 启动项目：hexo s

4. 添加第一篇文章

👉Hexo官方:写作

  1. 创建新文章：hexo n 文件名
  2. 清理：hexo clean
  3. 生成文章：hexo g
  4. 重启：hexo s

5. 部署到GitHub上

5.1. 创建git仓库,名称为 “git账号名.github.io”
5.2. 安装github部署插件。
在本地博客路径下，运行 cnpm install –save hexo-deployer-git
